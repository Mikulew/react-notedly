## General info

Notedly is project using React, MongoDB, Node.js, Express and GraphQL. It's a simple note Web Application that can support CRUD operations.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- React
- GraphQL
- MongoDB
